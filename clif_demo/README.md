# clif_demo — bundled CLIF demo dataset

Synthetic CLIF tables (**no real patient data**) shipped so the templates run out of the box.

The project config (`config/config.json`) points `data_directory` here by default — change it to your
site's CLIF tables directory for a real run (see [`config/README.md`](../config/README.md)).

Source: the CLIF demo dataset distributed with
[clifpy](https://github.com/Common-Longitudinal-ICU-data-Format/clifpy).
